Academic Project

MILO. eCommerce Site

Date: Spring 2019

Duration: 3 weeks

Team: Midhat Kazmi

Role: Research, Visual Design, Web Development

The following work is a personal redesign of a previous academic project.

Context

This is a school project to design an interactive eCommerce website. This project was mainly focused on front-end web technology. I used HTML, CSS, and JavaScript to developed an eCommerce site dedicated to selling Scandinavian design furniture.

Brainstorm & Establish Style Guide

The first part of the project was brainstorming, I came up with an idea to create a website for a Scandinavian design furniture company. In order to create a Scandinavian brand direction, I used a simple color scheme which is the combination of monochromatic brown colors, black and white. To create a contrast between the title and the body text, for headers, I used a sans-serif font - Roboto and for the content used Raleway to make users comfortable to read the text.

Exploration

After the design elements are ready, I drew a site map about the structure and navigation model of the website. As the sitemap is shown below, each page is coordinated with different functionalities.

Development process

The main goal of the project is to create a responsive website, therefore, I have to make sure the site is responding well on various screen sizes and devices (mobile to desktop).

Reflection

For next steps, I would like to add animations to enhance the interactions cue for interactive elements. Also, develop the back-end system in order to build a completely functional website.
